Intro to Unity w/ GDC

September 17, 2024 | 7:00 PM - 8:00 PM
In Person: AVW3258

Ever wanted to learn how to make your own games? Wanna brush up on the skills needed for making virtual-reality or augmented reality games/applications? Look no further than this workshop hosted in collaboration with the Game Developers Club! You'll learn the basics of navigating Unity, its core concepts, and some basic coding skills. We recommend that you have Unity Hub and a version of Unity installed before the workshop by downloading it from [their website](https://unity.com/download); we'll be using the FPS Microgame template. You should also have a code editor/IDE installed before the workshop begins, and Unity can automatically install Visual Studio--a popular IDE for game development--for you. Rider is also a fantastic option (coincidentally, my favorite option too, not biased at all), which students can get for free from [JetBrain's website](https://www.jetbrains.com/community/education/#students) and the [GitHub Student Developer Pack](https://education.github.com/pack).

Intro to Unity Shaders

April 23, 2024 | 5:00 PM - 7:00 PM
In Person: AVW 3258

Have you ever been dazzled by the visual art style of a video game? Have you wondered how video games are able to create beautiful and intricate visual effects, all while running smoothly? What’s so special about a graphics card, and why does my computer need one? In this workshop, we will discuss the answers to these questions while you learn the basics about computer graphics, shaders, and how they power video games and other real-time graphics applications. Then, you will employ those basic concepts to create your very own visual effects through Unity’s shader graph. No prior experience is required for this workshop, we welcome any and all who are interested!
